Backend Roadmap for Students

Your step-by-step guide to becoming a backend developmer. Start with the fundamentals and build your way up.

"Start small, Build big, Never stop!"

1. Languages

To buid the server-side of an application

  • Python Foundation
  • PHP Must Have
  • Javascript(Node.js) Must Have
  • Java Must Have
  • Ruby Must Have
  • Go Must Have

2. Frameworks (Git)

To help you build server-side apps faster and more easily.

  • Express.js Must Have
  • Django Optional
  • Laravel Optional
  • Spring Boot Optional
  • Flask Optional

3. Database

Systematically stores data and allows reading, updatting and deleting the data as needed.
  • MySQL Must Have
  • PostgreSQL Optional
  • MongoDB Optional
  • Redis Optional

4. APIs

A system that lets software talk to each other share data safely and simply.

  • RESTful API Must Have
  • GraphQL Advanced

5. DevOps

Working together to build and deliver backend software faster and more reliably.

  • Docker Must Have
  • GitHub Action Optional
  • Jenkins Optional
  • Nginx Optional
