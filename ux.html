UX Design Roadmap for Students

Your step-by-step guide to becoming a UX design developmer. Start with the fundamentals and build your way up.

1. UX Designer

User experience planning, wireframing, usability testing.

  • Research Foundation
  • Wireframing Must Have
  • Prototyping Must Have

2. UX Researcher

Conducts user research, interviews, surveys.

  • Interviewing Must Have
  • Data Analysis Optional

3. Information Architecture

Organizes content and structure.

  • Sitemaps Must Have
  • Content Hierarchy Optional

4. UX Writer

Writes microcopy, button texts, error messages.

  • UX Writing Must Have
  • Content Design Must Have

5. Usability Analyst

Evaluates user-friendliness, runs usability test.

  • Usability Testing Must Have
  • Reporting Optional

6. Product Designer

End-to-end product experience, often includes both UX & UI.

  • Cross-functional Must Have
  • Strategy Optional

7. Visual Designer

Focuses on branding, visuals and aesthetics.

  • Graphic Design Must Have
  • Color Theory Must Have

8. Accessibility Specialist

Ensures design is inclusive for all users.

  • WCAG Guidelines Must Have
  • Screen readers Advanced
